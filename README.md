## Installation and Setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/niwayandm/porto-react.git
   cd porto-react
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npm run dev
   ```

4. **Build for production**

   ```bash
   npm run build
   ```
